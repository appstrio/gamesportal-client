Games Portal Client
=====================

## Using this project

1. Clone the repo

2. Install dependencies
```bash
$ npm install
```
3. Launch `gulp`, which builds files, watches them, runs a server at localhost:8080 (default)

4. Open browser at http://localhost:8080

5. $$$
